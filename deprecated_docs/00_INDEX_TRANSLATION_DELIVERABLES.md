## Quality Assurance Checklist

### Before Full Translation
- [ ] All translators have reviewed appropriate sample files
- [ ] Translation guidelines understood
- [ ] Test translations completed (5-7 questions per language)

### During Translation
- [ ] Consistency checks across similar questions
- [ ] Register/formality level maintained throughout
- [ ] Diacritical marks preserved
- [ ] No English left in translations

### After Translation
- [ ] All 238 questions translated per language
- [ ] UTF-8 encoding validated
- [ ] RTL languages (ar, ur) tested for display
- [ ] Special characters verified
- [ ] Native speaker review completed
- [ ] Psychological soundness validated
- [ ] Cultural appropriateness confirmed

### Before Deployment
- [ ] JSON syntax validated
- [ ] All language codes present in metadata
- [ ] Database schema updated
- [ ] API endpoints tested with all languages
- [ ] Language switching tested in UI
- [ ] Performance validated with multi-language data
